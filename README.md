Calculator_Python
=================

A calculator based on python.

# requirement:
- Python 3.x
- PyQt5

# Installation
1. Clone this repository:
```sh
git clone  https://github.com/comtru/Calculator_Python.git
```

2. Installation dependencies:
```sh
pip install pyqt5
```

# Use
```sh
python Calculator.py
```
